# <a name="appendix-a---azure-rtos-netx-secure-returnerror-codes"></a>Приложение A. Коды возврата и ошибок NetX Secure (ОСРВ Azure)

## <a name="netx-secure-tls-return-codes"></a>Коды возврата NetX Secure TLS

В таблице 1 ниже перечислены возможные коды ошибок, которые возвращаются службами NetX Secure TLS (ОСРВ Azure). Обратите внимание, что эти службы могут возвращать и коды ошибок TCP/IP. Все коды TLS начинаются с 0x101, а номера кодов TCP/IP не превышают 0x100. Возвращаемые значения X.509 начинаются с кода 0x181. Сведения о возвращаемых значениях TCP/IP см. в документации по TCP/IP в NetX, а сведения о значениях X.509 приводятся ниже.

| NX_SECURE_TLS_UNSUPPORTED_CIPHER                  | 0x106  | При выполнении шифрования или расшифровки выбранный шифр был отключен или недоступен.                                                                           |
